#### [ReWrapped Spotify](https://github.com/jprkindrid/rewrapped-spotify)
**Full-stack Go/React application** to analyze and visualize years of exported Spotify data.

![Go](https://img.shields.io/badge/Go-00ADD8?style=flat-square&logo=go&logoColor=white)
![React](https://img.shields.io/badge/React-61DAFB?style=flat-square&logo=react&logoColor=black)
![SQLite](https://img.shields.io/badge/SQLite-003B57?style=flat-square&logo=sqlite&logoColor=white)
![Cloudflare](https://img.shields.io/badge/Cloudflare_R2-F38020?style=flat-square&logo=cloudflare&logoColor=white)
![Docker](https://img.shields.io/badge/Docker-2496ED?style=flat-square&logo=docker&logoColor=white)

- **Architecture:** Engineered a data pipeline utilizing local **SQLite** for development and **Turso (LibSQL)** for distributed production storage.
- **Storage:** Integrated **Cloudflare R2** for efficient JSON blob storage and implemented secure **OAuth2** authentication.
- **API:** Built a high-performance REST API in Go featuring pagination, summary statistics, and complex filtering.

#### [Stream-Linkify](https://github.com/jprkindrid/Stream-Linkify-Backend)
**Cross-platform music link converter** that turns track URLs into universal links (Spotify, Apple Music, TIDAL).

![.NET 10](https://img.shields.io/badge/.NET_10-512BD4?style=flat-square&logo=dotnet&logoColor=white)
![TypeScript](https://img.shields.io/badge/TypeScript-3178C6?style=flat-square&logo=typescript&logoColor=white)
![Azure](https://img.shields.io/badge/Azure-0089D6?style=flat-square&logo=microsoftazure&logoColor=white)
![Tailwind](https://img.shields.io/badge/Tailwind_CSS-38B2AC?style=flat-square&logo=tailwindcss&logoColor=white)

**Repositiories:** [Backend (C#)](https://github.com/jprkindrid/Stream-Linkify-Backend) | [Frontend (React)](https://github.com/jprkindrid/stream-linkify-frontend)

- **Search Logic:** Implemented **ISRC/UPC normalization** with fallback search to ensure accuracy across streaming providers.
- **Backend:** Deployed a **.NET REST API** to **Azure App Service** with automatic token refresh flows for Spotify and Apple Music.
- **Frontend:** Built a React UI featuring dynamic theming that extracts dominant album-art colors using the **OKLCH (culori)** color space.
